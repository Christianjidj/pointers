# pointers


¿Qué fue lo más importante que aprendiste sobre apuntadores?
Lo versatiles que son. Pode usar apuntadores para metodos dentro de una clase, o para una clase directamente.


¿Qué diferencia hay entre acceder con . y acceder con ->?
Justo es el caso que mencione en la pregunta anterior. Usar -> es llamar un metodo a traves de un puntero que apunta a un objeto. Mientras que . es el metodo convencial AAA.Get():


¿Por qué es importante liberar memoria dinámica?
Segun lo que entendí, es para hacer mas eficiente el codigo. Resulta util con codigos muy grandes.
